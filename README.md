# 🔢 Bin2Dec - Conversor de Binário para Decimal

Este projeto é uma ferramenta de conversão de bases numéricas que transforma sequências de números binários (base 2) em números decimais (base 10). Desenvolvido como parte do meu portfólio de **Ciência da Computação**, o foco principal foi a implementação manual da lógica de conversão para demonstrar o domínio sobre algoritmos de baixo nível e aritmética computacional.

## 🚀 Funcionalidades

* **Validação de Entrada:** O sistema identifica em tempo real e barra qualquer caractere que não seja `0` ou `1`.
* **Cálculo Manual:** Processamento via laço de repetição que percorre a string binária e aplica as potências de 2, demonstrando a lógica por trás da conversão de bases.
* **Interface Modern Tech:** Layout estruturado com **Flexbox**, focado em *Dark Mode* para melhor conforto visual e usabilidade (UX).
* **Responsividade:** Totalmente adaptado para dispositivos móveis e desktops.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessibilidade.
* **CSS3:** Estilização avançada, variáveis de cores e animações de interação (`hover`, `active`).
* **JavaScript (ES6+):** Manipulação do DOM, tratamento de erros e lógica algorítmica.
